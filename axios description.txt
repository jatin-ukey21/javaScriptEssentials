### **What is Axios?**
Axios is a **promise-based HTTP client** for making HTTP requests in JavaScript. It works both in the **browser** and **Node.js** and provides a simple API to fetch data from APIs, send data to servers, and handle requests efficiently.

## **Basic Axios Example (GET Request)**
```javascript
axios.get('https://jsonplaceholder.typicode.com/posts/1')
    .then(response => {
        console.log('Data:', response.data);
    })
    .catch(error => {
        console.error('Error:', error);
    });
```
ğŸ”¹ Axios automatically converts the response to JSON.  

---

## **POST Request with Axios**
```javascript
axios.post('https://jsonplaceholder.typicode.com/posts', {
    title: 'Hello World',
    body: 'This is a sample post',
    userId: 1
})
.then(response => console.log('Response:', response.data))
.catch(error => console.error('Error:', error));
```
ğŸ”¹ We pass an object as the request body.  
ğŸ”¹ Axios sets the `Content-Type: application/json` automatically.  

---

## **Using Axios with Async/Await**
```javascript
async function fetchData() {
    try {
        const response = await axios.get('https://jsonplaceholder.typicode.com/posts');
        console.log(response.data);
    } catch (error) {
        console.error('Error fetching data:', error);
    }
}

fetchData();
```
ğŸ”¹ **Async/Await** makes it easier to read and handle asynchronous requests.  

---


## **Handling Multiple Requests (`axios.all`)**
```javascript
axios.all([
    axios.get('https://jsonplaceholder.typicode.com/posts/1'),
    axios.get('https://jsonplaceholder.typicode.com/posts/2')
])
.then(axios.spread((post1, post2) => {
    console.log('Post 1:', post1.data);
    console.log('Post 2:', post2.data);
}))
.catch(error => console.error(error));
```
ğŸ”¹ `axios.all()` is useful for **parallel requests**.
